{Function} implementIn(oTarget, bAllowAbstract)
implementIn is the method to be used when implementing an interface in one class (oTarget) When the interface is implemented the method creates a callback that will throw an error if the method is not overwritten. Also adds a check to the constructor of the oTarget to know if all the methods are implemented. This check by default throws an error if one of the interface methods is not implemented on the class which implements the interface. If you want to implement an interface but you want to return an abstract class you will need to pass a boolean (true).
